The kube-assist operator is now running in namespace {{ .Release.Namespace }}.

To create a TroubleshootRequest:

  apiVersion: assist.cluster.local/v1alpha1
  kind: TroubleshootRequest
  metadata:
    name: my-troubleshoot
    namespace: default
  spec:
    target:
      kind: Deployment
      name: my-deployment
    actions:
      - diagnose
      - logs
      - events

To create a TeamHealthRequest:

  apiVersion: assist.cluster.local/v1alpha1
  kind: TeamHealthRequest
  metadata:
    name: my-health-check
    namespace: default
  spec:
    namespaces:
      - default
      - my-app
